# Paillard Sirene

Sirene is a Node js project for high perf data with no sql.

## Installation

Use the package manager [YARN](https://yarnpkg.com/) to install Yarn.

Use the db mongo [MongoDb](https://www.mongodb.com/) to install Mongo.

```yarn
yarn install 
```

## Usage

```start
nvm start

# For pause
nvm pause

# For unpause
nvm pause

nvm clean
for clean data base
```
